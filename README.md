# FloWedding — Undangan Digital (React + Parallax)

Project undangan digital ReactJS dengan efek parallax + reveal animation, siap dijalankan.

## Jalanin

```bash
npm install
npm run dev
```

Buka `http://localhost:5173`.

## Customisasi cepat

- Edit data utama (nama mempelai, tanggal, lokasi, agenda, WA) di `src/data/invitation.js`
- Nama tamu via URL: `http://localhost:5173/?to=Nama%20Tamu`
- Link Maps: isi `mapsUrl`
- Agenda acara: isi `event.agenda`
- File utama: `/FloWedding/src/App.jsx` dan style: `/FloWedding/src/styles.css`.

## Build

```bash
npm run build
npm run preview
```
