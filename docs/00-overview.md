# Overview

Digital wedding invitation berbasis React dengan animasi parallax untuk pengalaman undangan yang interaktif di smartphone.

Tujuan:
- membuat undangan digital yang menarik secara visual
- mudah dibagikan melalui web link
- memberikan pengalaman storytelling kepada tamu undangan

Target user:
- pasangan pengantin
- tamu undangan yang membuka link invitation

Main stack:
- React

Application flow:
Guest Open Link
→ Cover Header Invitation
→ Scroll Animation / Parallax
→ Couple Information
→ Event Section
→ Story Section
→ RSVP Section
→ End Section
