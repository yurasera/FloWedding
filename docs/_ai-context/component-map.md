# Component Map

Peta komponen utama dan perannya.

- App: komposisi halaman, state UI global kecil.
- CoverHeader: layar pembuka + dekorasi + trigger open.
- TopNav: navigasi anchor.
- Section: wrapper section dan background.
- CoupleSection: info mempelai + reveal/parallax.
- EventSection: detail acara + modal kalender.
- StorySection: cerita singkat.
- RsvpSection: form RSVP + CTA.
- Footer: penutup.
- Modal: dialog overlay.
- Reveal: animasi masuk saat terlihat.
- ParallaxLayer: parallax berbasis scroll.
- MicroParallaxScene: dekorasi mikro (dotgrid/sparkles/florals/ivy).
- CloudParallaxScene: dekorasi awan.
- Countdown: hitung mundur.
- RSVPForm: form RSVP + localStorage + WhatsApp link.
